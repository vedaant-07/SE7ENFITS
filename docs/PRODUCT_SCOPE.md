# SE7ENFITS Product Scope

SE7ENFITS is an AI-powered gym ecosystem for Indian gym members and gym owners.

## Product split

1. Member mobile app
2. Gym owner web dashboard
3. SE7ENFITS admin web dashboard
4. Shared backend API

The member app is for users only. Gym owners and admins use separate websites.

## Member app

Core modules:

- Google login
- Member onboarding
- Profile setup: name, DOB, gender, height, weight, location, fitness goal, activity level, experience level
- Gym code linking
- Subscription plans
- Daily workouts
- Weekly workout planner
- Exercise library
- Animated workout demos and videos
- Workout tracking: sets, reps, weight, rest, calories, PRs
- Food scan calorie detector
- Manual calorie calculator
- AI trainer for gym/workout/nutrition questions only
- Gym section showing linked gym, machines and trainers
- Challenges
- Reward points
- Leaderboards
- Progress tracking
- Notifications

## Gym owner dashboard

Core modules:

- Google login
- Gym profile
- Gym code and QR
- Member management
- Subscribed and non-subscribed members
- Membership tracking
- Trainer management
- Machine management
- Maintenance tracking
- Attendance
- Communication center
- Revenue analytics
- 25 percent commission ledger
- Payout history
- Gym leaderboard

## Admin dashboard

Core modules:

- All users
- All gyms
- Gym owners
- Subscriptions
- Payments
- Commissions
- Payout approvals
- Reward withdrawals
- Coupons
- Challenges
- Exercise library
- Food database
- AI settings
- Fraud review
- Audit logs

## Main business workflow

1. Gym owner creates gym profile.
2. Gym owner receives a unique gym code.
3. Gym owner shares code or QR with gym members.
4. Member signs in with Google and enters gym code.
5. Member buys SE7ENFITS subscription.
6. Member gets linked to that gym.
7. Gym owner earns commission on successful subscription.
8. Member workouts are generated using that gym's available machines.

## Unique value

SE7ENFITS AI trainer creates workouts based on the exact machines available in the user's linked gym.
